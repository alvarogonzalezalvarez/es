About Me

My name is Álvaro González Álvarez and I'm a Web Developer from Jerez de la Frontera (Spain). I started my career as a Java EE Developer right after finishing my studies in 2008. Since then, I have developed several enterprise web applications using many Java frameworks and, more recently, I have been working on the front-end and back-end sides of e-commerce oriented projects, which enabled me to get the IBM WebSphere Commerce certification.

I'm also a self-taught person who likes to learn, practice and use new booming JavaScript technologies.
